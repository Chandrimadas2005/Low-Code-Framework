# 🚀 Low-Code ML Framework (Unified System)

A unified low-code machine learning framework that supports both structured datasets and time series data.

---

## 🚧 Project Status (Important)

⚠️ **This project is currently under development**

* The **Structured Data Module** and **Time Series Module** are being developed in separate branches
* Integration into the main branch is **not yet completed**
* Final unified pipeline will be available after merging both modules

---

## 📌 Overview

This project combines two modules:

### 🔹 Structured Data Pipeline *(In Progress)*

* LLM-based feature selection
* Automated EDA
* AutoML model training

### 🔹 Time Series Pipeline *(In Progress)*

* Time series EDA
* Data cleaning
* Forecasting models

---

## 🔄 Planned Unified Workflow

```text id="r7k1mz"
Input Dataset
     ↓

     ↓
├── Structured Data (Under Development)
│     → Chatbot + LLM processing
│     → EDA + preprocessing

│
└── Time Series Data (Under Development)
      → Time series EDA
      → Data cleaning
      → Forecasting models
      → Prediction & evaluation
```
